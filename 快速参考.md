# 题目质量验证 - 快速参考

## 快速开始

### 1. 基本使用

```python
from core.question_validator import QuestionValidator

# 创建验证器
validator = QuestionValidator()

# 验证单题
question = {
    'correct_word': 'success',
    'options': ['success', 'succeed', 'successful', 'successfully'],
    'pos': 'noun',
    'context': 'Hard work leads to success.',
    'position': 4
}

result = validator.validate_question(question)
print(f"评分: {result['score']}/10")
print(f"有效: {result['is_valid']}")
```

### 2. 批量验证

```python
# 批量验证
questions = [q1, q2, q3]
batch_result = validator.validate_batch(questions)

# 生成报告
report = validator.generate_validation_report(batch_result)
print(report)
```

### 3. 与出题引擎集成

```python
from core.exam_generator import ExamGenerator

generator = ExamGenerator()
result = generator.generate_exam(text, num_blanks=10)

# 整体评分
print(f"整体评分: {result['overall_score']}/10")

# 每道题评分
for q in result['questions']:
    print(f"第{q['number']}题: {q['quality_score']}/10")
    if q['validation']['issues']:
        print(f"  问题: {q['validation']['issues']}")
```

## 验证结果解读

### 返回格式

```python
{
    'is_valid': bool,      # 是否有效（无严重问题）
    'score': float,        # 质量分数 (0-10)
    'issues': list,        # 严重问题列表
    'warnings': list,      # 警告列表
    'details': dict        # 详细验证结果
}
```

### 评分标准

| 分数 | 等级 | 说明 |
|------|------|------|
| 9-10 | 优秀 | 符合中考标准 |
| 7-9 | 良好 | 基本符合 |
| 5-7 | 一般 | 需要关注 |
| 0-5 | 较差 | 需要改进 |

## 常见问题处理

### Q: 如何忽略某些验证？
```python
# 在调用时过滤结果
result = validator.validate_question(question)
if result['score'] >= 8.0:  # 只使用高分题目
    accept_question(result)
```

### Q: 如何自定义权重？
```python
validator = QuestionValidator()
validator.SCORE_WEIGHTS = {
    'pos_accuracy': 0.40,      # 增加词性权重
    'distractor_quality': 0.30,
    'answer_uniqueness': 0.20,
    'blank_rationality': 0.05,
    'difficulty_match': 0.05
}
```

### Q: 如何添加同根词映射？
```python
# 在 validate_distractor_consistency 方法中
# 添加到 root_word_mappings 字典
'new_word': ['new_word', 'new_word_form1', 'new_word_form2']
```

## 验证规则详解

### 1. 词性标注验证
- ✅ 单词词性与声称一致
- ✅ 支持 NLTK 增强（如已安装）
- ⚠️ 无 NLTK 时使用词尾规则

### 2. 干扰项一致性
- ✅ 干扰项是真实单词
- ✅ 干扰项词性与答案一致
- ✅ 干扰项无重复
- ✅ 干扰项与答案不同

### 3. 答案唯一性
- ✅ 干扰项不在上下文中
- ✅ 答案是唯一合适选择

### 4. 挖空位置
- ✅ 不是专有名词
- ✅ 不是数字
- ✅ 不是单字母词
- ⚠️ 功能词可能有警告

### 5. 难度匹配
- ✅ 词汇难度适中
- ⚠️ 偏离目标难度有警告
