## 🎯 核心功能

### 验证规则

| # | 规则名称 | 功能 | 权重 |
|---|---------|------|------|
| 1 | 词性标注验证 | 验证单词词性是否正确 | 30% |
| 2 | 干扰项一致性 | 检查干扰项质量 | 25% |
| 3 | 答案唯一性 | 确保答案唯一确定 | 25% |
| 4 | 挖空合理性 | 检测挖空位置 | 10% |
| 5 | 难度匹配 | 评估难度是否适中 | 10% |

### 质量评分算法

```
总分 = 词性准确性(30%) + 干扰项质量(25%) + 答案唯一性(25%) + 挖空合理性(10%) + 难度匹配(10%)
```

## 🎓 使用示例

### 基本使用

```python
from core.question_validator import QuestionValidator

validator = QuestionValidator()
result = validator.validate_question({
    'correct_word': 'success',
    'options': ['success', 'succeed', 'successful', 'successfully'],
    'pos': 'noun',
    'context': 'Hard work leads to success.',
    'position': 4
})

print(f"评分: {result['score']}/10")
```

### 与出题引擎集成

```python
from core.exam_generator import ExamGenerator

generator = ExamGenerator()
result = generator.generate_exam(text, num_blanks=10)

print(f"整体质量评分: {result['overall_score']}/10")
```
